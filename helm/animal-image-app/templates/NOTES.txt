To access the application from a host with kubernetes, run the following command:
kubectl --namespace {{ .Release.Namespace }} port-forward service/{{ include "animal-image-app.fullname" . }} {{ .Values.service.port }}:{{ .Values.service.port }}
The application with then be available on http://127.0.0.1:{{ .Values.service.port }}

If using minikube, run the following command:
minikube service {{ include "animal-image-app.fullname" . }} --url
The above will provide a url in the format http://127.0.0.1:[EPHEMERAL PORT]
